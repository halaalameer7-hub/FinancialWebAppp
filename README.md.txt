# Financial Dashboard Web App

A simple web application to input Revenue and Expenses and automatically generate financial reports, including:

- Net Profit calculation
- Variances (Revenue, Expenses, Profit)
- Net Profit Margin
- Smart financial insights

## How to use

1. Open `index.html` in your browser.
2. Enter Revenue and Expenses.
3. Click "Submit" to see the report.

## Tech Stack

- HTML, CSS, JavaScript (Frontend)
- Python Flask (Backend, optional)
